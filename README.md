# 音楽同期歌詞ファイル作成ツール
ボカコレアプリ上などで表示する歌詞を音源と同期させることができるLRCファイルを作成するアプリ。

サーバー通信をしない動作なので、未発表曲でも安心して作成できます。

ブラウザ上で音源（MP3等）を聴きながら、歌詞と同期したタイムスタンプ付きのLRCファイルを作成できるツールです。

## 特徴
- アップロードした音源や歌詞データはブラウザ（ローカルメモリ）上でのみ処理されます。サーバーへ送信されることはありません。
- 音楽を再生しながらリアルタイムで「同期ポイント」ボタンを押すだけでLRCファイルが完成します。
- Webブラウザさえあればどこでも動作するのでインストール不要です。

## 使い方
1. 音源ファイルと歌詞テキストファイルを読み込みます。
2. 再生ボタンを押し、曲を聴きながら歌詞に合わせて「同期ポイントを記録」ボタンをクリックします。
3. すべての行にタイムスタンプがついたら、ファイル名を決めてダウンロードしてください。

## 制約・注意事項
プログラム開発に初挑戦ということもあり、以下の不便な点があります。あらかじめご了承ください。

- **改行の扱いについて**: 入力した歌詞ファイル内の空行（改行のみの行）は反映されません。LRCファイル出力後に、下記を参考にメモ帳などで改行とタイムスタンプを手動で追加して調整をお願いします。
```
[00:01:23]あいうえお
[00:01:23]
[00:01:26]かきくけこ
```
- **シーク機能の未実装**: 音源の操作は「再生」と「一時停止」のみです。同期をやり直したい場合は、お手数ですがページを更新して最初からやり直していただく必要があります。

「不便だな」と感じる部分もあるかと思いますが、シンプルな使い心地を優先して開発しています。
